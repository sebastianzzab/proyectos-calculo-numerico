# Proyectos de Cálculo Numérico

Este repositorio contiene una colección de proyectos y scripts relacionados con el cálculo numérico. Aquí encontrarás implementaciones de diversos algoritmos y métodos numéricos utilizados en el ámbito de la ciencia de datos y las matemáticas aplicadas.

Actualmente incluye:
- Implementación del método de Bisección para encontrar raíces de funciones.
- Implementación del método de Newton-Raphson para encontrar raíces de funciones.
