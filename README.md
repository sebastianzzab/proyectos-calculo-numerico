# Proyectos de Cálculo Numérico

Este repositorio centraliza los proyectos desarrollados para la asignatura de Cálculo Numérico, correspondiente a la Licenciatura en Estadística e Informática. Su objetivo es consolidar en un único lugar las implementaciones de los diversos algoritmos y métodos numéricos estudiados durante el curso, sirviendo como un portafolio académico y recurso de consulta.

## Contenido Actual

Actualmente, el repositorio incluye las siguientes implementaciones:
- Implementación del método de Bisección para encontrar raíces de funciones.
- Implementación del método de Newton-Raphson para encontrar raíces de funciones.
- Implementación del método de Integración Numérica de Riemann.
